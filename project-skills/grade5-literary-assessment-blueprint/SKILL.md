# Grade 5 Literary Assessment Blueprint

Read [references/blueprint.md](references/blueprint.md) before planning questions.

## Workflow

1. Read the complete assigned chapter before selecting evidence.
2. Mark 8-12 evidence candidates spanning beginning, middle, and end.
3. Fill all five blueprint slots; do not write duplicate skills about the same event.
4. Choose the Q5 rotation target specified for the chapter.
5. Record standard, evidence quote, intended reasoning, and DOK before drafting an item.
6. Reject a plan if two questions can be answered from the same isolated sentence unless they assess clearly different constructs.

## Output

For each planned item provide: `slot`, `standard`, `skill`, `dok`, `evidence`, and `intended_answer`.

## Gate

Do not proceed to item writing unless each item has exact chapter evidence and the five items collectively cover the full chapter rather than its opening pages only.
