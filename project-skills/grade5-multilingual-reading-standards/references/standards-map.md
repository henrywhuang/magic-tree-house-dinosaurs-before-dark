# Standards map

## Learner profile

- Grade: 5
- English status: multilingual/non-native, comparatively strong proficiency
- Text: chapter-based narrative fiction
- Design principle: grade-level cognition with concise, transparent assessment language

## Priority U.S. standards

| Code | Assessable reading action | Suitable selected-response evidence |
|---|---|---|
| RL.5.1 | Explain explicit meaning and draw an inference using accurate textual evidence | Select a stated detail, inference, or best supporting detail |
| RL.5.2 | Determine theme and summarize the text | Select a defensible theme or an essential-events-only summary |
| RL.5.3 | Compare/contrast characters, settings, or events using details | Explain motivation, response, relationship, similarity, or difference |
| RL.5.4 | Determine word/phrase meaning in context, including figurative language | Infer meaning or effect from the sentence and nearby context |
| RL.5.5 | Explain how chapters/scenes fit the overall structure | Explain how an event launches, develops, complicates, or resolves action |
| RL.5.6 | Describe how narrator point of view influences event description | Identify what the narration lets the reader know or feel |
| L.5.4a | Use context as a clue to word/phrase meaning | Infer an unknown or multiple-meaning word from local clues |
| L.5.5a-c | Interpret figurative language and word relationships | Interpret a simile/metaphor/idiom or a meaningful nuance |

## Multilingual-learner calibration

- Target WIDA Grades 4-5 Expanding to Bridging interpretive performance: follow event sequences, locate supporting details, answer analytical questions, infer word meaning from context, and draw conclusions from explicit and implicit information.
- Keep stems direct and normally 8-18 words, excluding quoted evidence.
- Prefer one clause in a stem; use two only when the evidence must be named.
- Keep options grammatically parallel and similar in length.
- Preserve important story language but avoid idiomatic distractors unrelated to the target.
- Allow cognitive challenge to come from evidence selection and reasoning, not from confusing syntax.
